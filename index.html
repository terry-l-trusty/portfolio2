Projects

Account Reconciliation Tool

  • Objective - Develop account reconciliation tool comparing thousands of trades across two databases. Display top variances and highlight driving factors behind the variance.

  • Process - Constructed Python script to download trade data from database. Compared and contrasted each position and generated an Excel file arranging data by largest variances.

  • Result - Reconciliation tool eliminated a majority of the manual data sourcing and reduces total team reconciliation time by 10%.

Profit & Loss Commentary Tool

  • Objective - Automate weekly commentary process, highlighting top Profit & Loss drivers and the underlying market conditions leading to week-over-week revenue movements.

  • Process - Scraped revenue data for each bond sector from the data lake. Generated commentary presenting top revenue drivers, distinguishing between new trades revenue and market movement revenue on existing inventory.

  • Result - Implemented the tool across different teams, reducing each individual team member workflow by 10%.

Inventory Management Tool

  • Objective - Automate the inventory management process particularly for those in the food hospitality industry. Provide an interface for the user to plan menus and track utilized inventory and receive alerts when items need to be re-stocked.

  • Process - Allow user to upload and select from an array of menu items with their costs, along with inventory movements as items are used. Send the user alerts when inventory gets below a certain point and make suggestions on which materials to buy.
